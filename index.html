Générique

Les formulaires sont décrits dans des feuilles de calculs dans le formalimsme XLSForm. Il est trés facile d'adapter des formulaires existants à ses propres besoins.

Intéropérable

L'API ODATA facilite l'intégration des données collectées à son système d'information. Avec Python (pyODK), R (ruODK), PostgreSQL (central2pg), avec un ETL comme fme, ou kettle, QGIS...

OpenSource / mu par ODK

ODK (Open Data Kit) est un outil opensource de génération de formulaires de collecte de données pour téléphones Android.
